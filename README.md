# Genuine Green Care

Thomas Marine–level marketing site for Genuine Green Care Landscaping.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Netlify (Forms + `@netlify/plugin-nextjs`)

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content to customize

- Business phone, email, cities: `src/data/site.ts`, `src/data/areas.ts`
- Services: `src/data/services.ts`
- Google reviews: `src/data/reviews.ts`
- Gallery projects: `src/data/gallery.ts`
- Blog posts: `src/data/blog.ts`

## Forms

Quote and contact forms submit to Netlify Forms via `public/__forms.html`. Enable Forms in the Netlify UI after deploy.
