# Genuine Green Care

Thomas Marine–level marketing site for Genuine Green Care Landscaping.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Netlify

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
